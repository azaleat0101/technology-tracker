# 🚀 Персональный трекер освоения технологий

SPA-приложение на React для отслеживания прогресса в изучении технологий по дорожным картам.

## 📋 Функциональность

- 📁 Загрузка дорожных карт из JSON файлов
- 📊 Визуализация прогресса с цветовыми индикаторами
- 📝 Добавление персональных заметок к темам
- 🎯 Управление статусами (Не начато/В работе/Выполнено)
- 📅 Установка сроков выполнения
- 💾 Автосохранение в localStorage
- 📤 Экспорт прогресса в JSON

## 🚀 Быстрый старт

```bash
# Клонирование
git clone <repository-url>
cd technology-tracker

# Установка зависимостей
npm install

# Запуск приложения
npm start
